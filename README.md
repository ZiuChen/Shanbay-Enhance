# Shanbay-Enhance

扇贝单词功能增强，带来更好的背单词体验。

## :sparkles: 已添加功能

* :heart: 单词总结释义遮挡（功能与手机端相同）
* :heart: 自定义单词/例句字体
* :heart: 自定义功能快捷键
* :heart: 专注模式/深色模式
* :heart: 隐藏例句翻译
* :heart: 更多功能请安装体验 ··· ···
